## 🚀 使用快速指南

### 初始化
```bash
# 激活虚拟环境
conda activate transformer_mt

# 确认数据准备完毕（已完成）
python download_dataset.py
```

### 训练
```bash
# 从头开始训练 (~1-2分钟)
python train.py

# 输出:
# - checkpoints/best_model.pt
# - checkpoints/last_model.pt
# - outputs/loss_curve.png
```

### 评估
```bash
# 完整评估和可视化
python evaluate.py

# 输出:
# - 测试损失值
# - BLEU分数
# - 注意力可视化 (12个PNG文件)
```

### 推理
```bash
# 交互式翻译
python inference.py

# 使用示例:
# Input: guten morgen .
# Output: good morning .
```

### 测试
```bash
# 运行完整测试
python test_complete.py

# 输出: 7项测试，全部通过 ✓
```
